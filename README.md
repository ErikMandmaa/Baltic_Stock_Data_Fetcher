# Baltic_Stock_Data_Fetcher
Program, that gives you instructions to download data from the Nasdaq Baltic Stock Exchange and analyzes the data (Monthly summaries: Open, High, Low, Close prices and volume, plus monthly returns and volatility Current metrics: Current stock price 30-day and 90-day volatility (annualized) Maximum drawdown Year-to-date return percentage).

What it actually does:

Provides clear instructions for manual download
Loads CSV files you download yourself
Tries Yahoo Finance as backup (for cross-listed stocks)
Offers sample data for testing code

The reality:

You need to visit nasdaqbaltic.com → find your stock → manually export the data
Or subscribe to Nasdaq Data Link (paid service)
Or use web scraping (usually against ToS)
